# Тест-кейсы для API микросервиса объявлений

## Общая информация
- **Base URL**: https://qa-internship.avito.com
- **API версии**: v1 и v2
- **Форматы данных**: JSON

## Техники тест-дизайна
- **Классы эквивалентности**: для валидации полей (sellerId, price, name)
- **Граничные значения**: для числовых полей
- **Попарное тестирование**: для комбинаций параметров
- **Предугадывание ошибок**: для негативных сценариев
